# OptiCrop

OptiCrop is a Machine Learning based crop recommendation system that predicts the most suitable crop using:

- Nitrogen
- Phosphorous
- Potassium
- Temperature
- Humidity
- pH
- Rainfall

## Technologies

- React
- TypeScript
- Flask
- Scikit-Learn
- Bootstrap
- SQLite

## Installation

npm install
npm run dev

Backend

python app.py
